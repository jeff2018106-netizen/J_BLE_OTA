## 📱 方案1: 蒲公英 (pgyer.com) ⭐⭐⭐⭐⭐ 推荐

### 步骤1: 注册账号（1分钟）

1. 打开 https://www.pgyer.com/
2. 点击 "注册" / 使用微信扫码注册
3. 登录后进入控制台

---

### 步骤2: 上传Xcode项目（3种方式）

#### 🅰️ 方式A: 直接上传Xcode源码（推荐）

```
1. 登录蒲公英控制台
2. 点击 "应用" → "上传应用"
3. 选择 "iOS应用"
4. 上传方式: 选择 "上传Xcode工程文件"
5. 选择整个 esp-ble-ota-ios 文件夹（或 .zip压缩包）
6. 点击 "开始上传"
7. 等待云端编译（约5-10分钟）
8. 编译成功！自动生成下载二维码
```

#### 🅱️ 方式B: 如果你有IPA文件

```
1. 控制台 → 应用 → 上传应用
2. 拖拽 .ipa 文件到上传区域
3. 或点击 "选择文件" 上传
4. 自动使用企业证书重新签名
5. 生成安装链接/二维码
```

#### ©️ 方式C: 使用API自动化上传（开发者推荐）

```bash
# 安装蒲公英命令行工具
# Windows (需要Python):
pip install pgyer

# 或使用curl直接上传:
curl -F "file=@你的IPA路径.ipa" \
     -F "_api_key=你的API Key" \
     https://www.pgyer.com/apiv2/app/upload
```

**获取API Key**: 
```
蒲公英 → 个人中心 → API信息 → 复制API Key
```

---

#### 方法1: 扫码安装（最简单）
```
编译完成后，页面会显示二维码 → 
iPhone相机扫码 → 
Safari浏览器打开 → 
点击 "安装" → 
输入锁屏密码确认 → 
完成！
```

可以发送给任何人:

```

#### 方法3: 通用安装链接
```

**答**: 够！每天10次上传，2个应用，对于开发测试绰绰有余。

### Q2: 企业签名安全吗？
**答**: 安全。蒲公英是正规平台，使用的是合法的企业开发者证书。

### Q3: IPA有效期多久？
**答**: 
- 免费用户: 7天（到期后重新上传即可）

**答**: 支持所有原生iOS APP：
- Objective-C / Swift
- React Native
- Flutter
- Unity
- Cocos Creator
- 等等...

### Q6: 上传失败怎么办？
**检查清单**:
- [ ] 文件大小 < 500MB（免费限制）
- [ ] Xcode版本兼容性（建议Xcode 13+）
- [ ] Bundle ID唯一性
- [ ] 网络连接正常
- [ ] 账号未违规

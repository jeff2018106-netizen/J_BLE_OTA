# 蒲公英 3分钟出包 - 极简操作手册

> **目标**: 上传代码 → 自动编译 → 扫码安装到iPhone  
> **前提**: 有网、有浏览器、有手机  
> **时间**: 首次5分钟，后续每次3分钟

---

## 🚀 第一步：注册账号（1分钟）

1. **打开浏览器**，访问：
   ```
   https://www.pgyer.com/
   ```

2. **点击右上角「登录/注册」**
   - 推荐用**微信扫码注册**（最快）
   - 或用手机号注册

3. **登录成功后**，进入控制台

✅ 完成！

---

## 📦 第二步：上传项目（2分钟）

### 方法A：上传Xcode源码（推荐新手）

```
1. 登录后，点击顶部菜单 「应用」→「上传应用」
2. 选择应用类型：「iOS 应用」
3. 上传方式选择：「上传源码」或「上传Xcode工程」
4. 点击「选择文件」或直接拖拽文件夹

   👉 选择这个文件夹：
      e:\Project\Dyna\Project\20241113\esp-ble-ota-ios\esp-ble-ota
   
   （或者先把这个文件夹压缩成zip再上传）
   
5. 填写基本信息：
   - 应用名称：esp-ble-ota（随便填）
   - 版本号：1.0.0
   
6. 点击「开始上传」或「确认上传」
7. 等待上传完成（取决于网速，通常1-3分钟）
```

### 方法B：如果你已经有IPA文件

```
1. 点击「上传应用」
2. 选择「iOS 应用」
3. 直接拖拽 .ipa 文件到上传区域
4. 或点击「选择文件」选取 .ipa 文件
5. 点击上传
6. 等待自动签名完成（约30秒-1分钟）
```

⚠️ **注意**：如果没有IPA文件，就用方法A上传源码。

---

## ⏳ 第三步：等待云端编译（3-10分钟）

上传成功后：

```
1. 页面会显示「正在编译」或「处理中」状态
2. 等待时间：3-10分钟（首次较慢，后续有缓存会快）
3. 可以刷新页面查看进度
4. 编译完成后状态变为「已发布」或「审核通过」
```

💡 **提示**：可以去做别的事，编译完成后蒲公英会通知你（如果配置了通知的话）。

---

## 📱 第四步：下载安装到iPhone（30秒）

编译成功后，你会看到：

### 方式1：扫码安装（最简单）⭐

```
1. 在应用详情页找到「二维码」（通常在右侧或底部）
2. 用iPhone相机扫描二维码
3. Safari浏览器自动打开
4. 点击「安装」按钮
5. 输入iPhone锁屏密码确认
6. 等待安装完成（约30秒-1分钟）
7. 桌面出现APP图标 → 完成！
```

### 方式2：链接分享

```
每个应用都有一个短链接，例如：
https://www.pgyer.com/abc123

使用方法：
- 微信/QQ发送给任何人
- 对方点击链接 → Safari打开 → 安装
- 或自己在iPhone Safari中打开此链接
```

### 方式3：直接下载IPA文件

```
1. 在应用详情页点击「下载」按钮
2. 保存 .ipa 文件到电脑
3. 之后可以用AltStore等工具安装（可选）
```

## ❓ 常见问题快速解决

### Q1: 上传失败？
**检查**：
- [ ] 文件大小 < 500MB
- [ ] 网络连接正常
- [ ] 文件格式正确（.zip 或 .ipa）
- [ ] 账号未违规

**解决**：压缩成zip后再试一次

### Q2: 编译失败？
**可能原因**：
- Xcode项目配置问题
- 缺少依赖库（CocoaPods等）

**解决**：
- 查看编译日志（页面会显示错误信息）
- 把错误信息发给我帮你分析

### Q3: 安装时提示"未信任"？
**解决**：
```
设置 → 通用 → VPN与设备管理 → 
找到开发者证书 → 点击信任 → 
输入锁屏密码 → 重启APP
```

### Q4: 7天后APP失效？
**正常现象**！免费版有效期7天。
**解决**：重新上传一次即可续期（30秒搞定）

### Q5: 想要更长时间有效？
- 升级VIP（约200元/年，有效期30天）
- 或申请自己的Apple Developer账号（$99/年，永久有效）
